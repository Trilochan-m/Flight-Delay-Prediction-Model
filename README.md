The Flight Delay Prediction System is a machine learning project designed to predict whether a flight is likely to be delayed based on weather conditions and flight schedule information. The project combines historical flight data with weather data to build an accurate predictive model that can assist airlines, airport authorities, and passengers in making informed decisions.

The application performs data preprocessing, feature engineering, and model training using the XGBoost algorithm to achieve reliable prediction performance. A user-friendly Flask web interface allows users to enter a flight ID and travel date to receive real-time delay predictions.

Features

- Predicts flight delays using weather and schedule data.
- Data preprocessing and feature engineering for improved model performance.
- XGBoost-based classification model for accurate predictions.
- Interactive Flask web application for easy user interaction.
- Clean and responsive interface for entering flight details.
- Efficient prediction pipeline for quick results.
